# 📚 Library Management System

A simple Java-based library system built for learning object-oriented programming and backend fundamentals.

---

## 🚀 Features
- Add books
- List books
- Search books
- Delete books

---

## 🛠️ Tech Stack
- Java 21
- Maven
- OOP Principles
- CLI (Console Application)

---

## 📁 Project Structure

```
src/
docs/
pom.xml
```
---

## ▶️ How to run

```bash
mvn clean install
java -jar target/library-management-system.jar
```

## 🎯 Purpose

This project was built to practice Java fundamentals, object-oriented design, and clean code structure.
